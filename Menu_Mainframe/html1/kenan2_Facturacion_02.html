Facturación Manual

Pasos para facturar.

1.- Conectarse al SRVFT51/User: ctmft.

1.1 Si la cuenta a facturar está en la Base de Datos CUS1, escribiremos la siguiente instrucción:

arbrun bip ac=40861924 db=kp01cus1 srv=3 prnum=54

Donde:

arbrun bip      es la instrucción para facturar

ac=40861924 va el número de cuenta a facturar

db=kp01cus1  Base de Datos

srv=3             Número del servidor

prnum=54      Número de proceso, cuando se inicie una nueva factura manual, incrementar en 1, quedaria asi prnum=55

1.2 Si la cuenta a facturar está en la Base de Datos CUS2, escribiremos la siguiente instrucción:

arbrun bip ac=42480028 db=kp01cus2 srv=4 prnum=31

Donde:

arbrun bip      es la instrucción para facturar

ac=42480028 va el número de cuenta a facturar

db=kp01cus2  Base de Datos

srv=4             Número del servidor

prnum=31      Número de proceso, cuando se inicie una nueva factura manual, incrementar en 1, quedaria asi prnum=32

1.3 Si la cuenta a facturar está en la Base de Datos CUS3, escribiremos la siguiente instrucción:

arbrun bip ac=1016192408 db=kp01cus3 srv=5 prnum=41

Donde:

arbrun bip          es la instrucción para facturar

ac=1016192408 va el número de cuenta a facturar

db=kp01cus3      Base de Datos

srv=5                 Número del servidor

prnum=41          Número de proceso, cuando se inicie una nueva factura manual, incrementar en 1, quedaria asi prnum=42

2.-Al finalizar, consultar la cuenta con la herramienta de Ronald y la información anexarla a la solicitud del analista de prueba.


3.-En el supuesto caso, se apague la máquina en plena facturación, proceda así:

Reconectarse al servidor SRVFT51, dar el siguiente comando:

arbrun sin db=kp01cus1 srv=3 prnum=83 una próxima vez incrementar en 1, quedaria asi prnum=84

arbrun sin db=kp01cus2 srv=4 prnum=200 una próxima vez incrementar en 1, quedaria asi prnum=201

arbrun sin db=kp01cus3 srv=5 prnum=100 una próxima vez incrementar en 1, quedaria asi prnum=101

NOTA Si se quiere facturar más de una vez, proceder según el CUS1, 2 o 3 con la isntrucción dada en los puntos 1.1, 1.2 y 1.3

Cambio de Password pass.bat

Cuando estamos consultando una cuenta Kenan con la herramienta de Ronald, puede ocurrir que se detanga y veras un recuadro que te indica que el password expiro, esto ocurre aproximadamente cada 30 días, proceder con los siguientas pasos:

1.-Buscar la carpeta Consultas/Pruebas en el disco C de tú PC.

2.- Buscar el archvo pass.bat Editarlo y observaras lo siguiente:

set passw05=ambi_pf84 Cambiar ambi_pf84 por ambi_pf85 solo incrementar en uno
set user05=amb05                  No cambiar

::set passint=ARBOR-,#ago12  No cambiar
 
:: paint =ambi_pf29 No cambiar
 
:: viejo pass de arbor Kenan_HPUX_2008 No cambiar

3.- Abrir TOAD y entrar a las Base de Datos KP01CUS1, KP01CUS2, KP01CUS3, KP01ADM, KP01CAT y KP01INT con el ususario arbor y escribiremos la siguiente instrucción en cada uno de ellos:

Desbloquear usuario
ALTER USER amb05 ACCOUNT UNLOCK;
Cambiar password
alter user amb05 identified by ambi_pf85

Git paso a paso
Proyecto Nuevo

git init
git add README.md (no se uso) usamos (git add .)
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/luisengardo60/Linux-ayuda.git
git push -u origin main

1.- Ir a la carpeta donde se tiene el proyecto
Abrir git batch

2.- git init (inicializar git en la carpeta del proyecto)

3.- git status (ver estado)

4.- git add .   (ingresar todos los archivos contenidos en una carpeta)

5.- git commit -m "Comentario"  (comprometer los elementos en el repositorio  local hacia la nube de git)

6.- git branch -M main (le estamos indicando a git que vamos a subir los elementos a la nube de la (branch)rama principal)

7.- git remote add origin https://github.com/luisengardo60/Linux-ayuda.git (adiconar al repositorio git de la nube)

8.- git push -u origin main (realiza la sincronizaciòn de los elementos del archivo local a la nube git,
                             solicitara usuario y password)

9.- Verificar en la nube git

Modificar Proyecto

1.- En git batch ubicarse en la ruta del proyecto

2.- En la linea de comando dar code . para abrir VSC realizar los cambios necesarios

3.- git status para ver los cambios realizados

4.- git add . (Adiconar los cambios)

5.- git commit -m "comentario" (Comprometer los cambios desde el repositorio local a la nube git)

6.- git push -u origin main (sincronizar local y nube git)

7.- git log (verifica los ùltimos commit realizados)

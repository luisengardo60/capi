Normas y procedimentos para el usuario de Ambiente

Reset

Cuando los analistas de pruebas nos solicitan resetear una clave en particular, procederemos así:

Ingresar al srvft51 con el usuario: admqa, ver siguiente ejemplo:

srvft51 38: sudo /usr/lbin/modprpw -k aindri

Para este caso se reseteara al usuario aindri y tendrá el mismo password que se le había otorgado
Usuarios que tienen los usuarios de pruebas:
nmarin, kbraba
El de ambiente es el ctmft



Mandar correo a todos los analistas de pruebas indicandoles la nueva clave del usuario solicitado
 


Reset y New

	
 Cuando los analistas de pruebas nos solicitan cambiar en particular, procederemos así: 

Ingresar al srvft51 con el usuario: admqa, ver siguiente ejemplo:

srvft51 38: sudo /usr/lbin/modprpw -k aindri desbloquea

Luego

srvft51 40: sudo /usr/bin/passwd aindri solicitará nuevo password debe tener 8 caracteres


NOTA Mandar correo a todos los analistas de pruebas indicandoles la nueva clave del usuario solicitado.
